Data Analysis

Please provide the direct download link of the subtitle file.
Our system will automatically uncompress and analyze the srt file and suggest the language difficulty.

For example, find movie at "https://subscene.com/subtitles/inside-out-2015/english/1206103" and there's a green button "Download English Subtitle" on the page.
The only thing to do next is to right-click the button and select "Copy Link Address". Paste the link to the text area here.
Note there's certain limitation to the Java Util.Zip library so not all the zip files are supported.
